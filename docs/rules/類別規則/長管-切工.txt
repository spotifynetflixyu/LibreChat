【長條料/切工類別規則】

一、適用範圍
適用鐵軌、H型鋼、I型鋼/工字鐵、角鐵、槽鐵、平鐵、圓鐵、方鐵、圓管、方管、扁方管等需裁切長條料。

二、材料是否切清
槽鐵、角鐵、鐵軌、圓管、圓鐵、平鐵、扁方管、方管預設「不切清」，優先採支、只、片等相符整件 direct 單價。H型鋼、方鐵以 Kg 為主，只有訂單明確指定「不切清」時，才改採 M、支或其他整件 direct 單價；找不到可用 Kg 價時送人工複核，不自動改用其他單位。其餘長條料除非明確說「可切清、切清賣、只算成品長度」，否則維持「不切清」。
不切清時，材料費依素材長度配料計算，需列素材長度、素材支數、可裁成品數、餘料長度/重量。素材長度不明時，用規則預設；仍不明暫用 6M 並標低信心。鐵軌若資料指定 10M，優先用 10M。

三、切工基本價查法
`search_price_candidates` 先完成全部一般價格 queries，之後 backend 統一查一次 `steel.cutting_prices`，並把結果放在頂層 `cuttingPrices`。AI 不另建切工 query。

切工查詢只使用類別包含條件，不使用厚度、尺寸、keyword 或 limit；同一類別的所有規格、空白價格與補充規則都回傳給 AI 判斷。資料表的 inch 已用 1 inch = 25.4mm 精確換算並保留小數，但不作 query filter。

自動對應：

| 材料 category | 切工包含搜尋詞 |
| --- | --- |
| H型鋼 | H型鋼 |
| 圓管、方管、扁方管 | 鐵管 |
| 角鐵 | 角鐵 |
| 槽鐵 | 槽鐵 |
| 平鐵 | 平鐵，可匹配 `鐵板/平鐵` |
| 鐵板 | 鐵板，可匹配 `鐵板/平鐵` |

`cuttingPrices[].queryIds` 使用 backend 依 `queries` 順序產生的 `q1`、`q2`…，並與 `sourceCategories` 一起對回原訂單列。I型鋼/工字鐵、鐵軌、圓鐵、方鐵、板/浪板不在自動切工對應範圍；若另有明確 ERP 加工品項，才以一般 `加工/切工` price query 查詢，不可自行套用其他類別。

四、基本公式
切工費 = (基本切工 direct 單價 + 加價) × 刀數 × 數量基準。
若價格列已含加價或為式價，依 price row/category rule 單位優先，不得重複加價。價格空白、0、未確認或待確認者不可填 0，列人工複核。

五、刀數計算
一個切口 = 1 刀。

未修頭尾時：

- 成品長度 = 素材長度，且不需裁切：0 刀
- 一支素材切成 n 支且無餘料：n − 1 刀
- 一支素材切成 n 支且有餘料：n 刀
- 不易逐支判斷時，可用成品支數暫估刀數，但需標註「切刀數以成品支數估算」

切頭/切尾：

- 切頭：每支素材 +1 刀
- 切尾：每支素材 +1 刀
- 切頭尾：每支素材 +2 刀
- 若明確說每支成品修頭尾，則每支成品 +2 刀
- 對半切不修頭尾 = 1 刀
- 對半切 + 修頭尾 = 3 刀

六、額外加價/限制規則

H型鋼/I型鋼/工字鐵：

- H型鋼斜切：切平行斜刀 = 基本價 ×2 −10；切梯形斜刀 = 基本價 ×2；翼板切斜 = 基本價 ×2 +50
- H 14mm 以上另計：開槽 KZZB10 A/C/F=140、B=150；沖孔 KZZB11 A/C/F=16、B=17；倒角 KZZB12 A/C/F=140、B=150
- H型鋼套用工字鐵切工價格時，價格再加50
- 大料、加厚、特殊角度、翼板切斜註記不清時列人工複核

管類：

- 黑鐵管類切斜 = 基本價 ×2 −5
- 管類厚度厚另計
- 方管厚度 1.2 以下不切
- 白A、錏方管 +5
- 白鐵 100 以下 +5；白鐵 100 以上 +10
- 1吋以下小方管量少加價
- 圓條預設不切；1吋以下圓條不切或需外切，列人工複核

角鐵：

- 白鐵角鐵在黑角鐵基本切工上 +5
- 材質或尺寸不清楚，標低信心

槽鐵：

- 特長、修頭尾外 +30
- 特短外 +10，暫以 500mm 以內判斷
- 特長/特短界線不明時列人工複核

平鐵：

- 白鐵平鐵另計
- 1吋以下平鐵量少加價
- 寬度不明時不可自行補值；厚度未指定時依通用規則，在其他條件相符且可報價的候選中採最小厚度。若無法唯一確認候選，列人工複核。

七、低信心條件
以下不可填 0，需標低信心或人工複核：價格空白、價格為 0、手寫不清、材質不明、未指定厚度且無法依通用最小厚度規則唯一選價、規格無完全匹配、素材長度不明、切清/不切清不明、修頭尾是素材修或成品修不明、特殊角度或斜切方式不明。
